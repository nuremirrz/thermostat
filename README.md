# Симулятор подключения термостата

Интерактивная 3D-сцена: пользователь снимает термостат со стены и подключает
4 провода к нужным клеммам задней панели. Реализовано на **Three.js + TypeScript + Vite**.

## Запуск

```bash
npm install
npm run dev      # дев-сервер на http://localhost:5173
npm run build    # production-сборка в dist/
npm run preview  # предпросмотр собранной версии
```

## Сценарий уровня (по ТЗ)

1. На стене закреплён термостат (модель `thermostat.glb`).
2. Клик по термостату — он отъезжает в сторону и открывает заднюю панель.
3. Видны панель и 4 провода, выходящие из стены: зелёный, жёлтый, белый, красный.
4. Провода не подключены. Игрок выбирает провод, затем кликает по клемме:
   - Зелёный → **G**
   - Жёлтый → **Y**
   - Белый → **W**
   - Красный → **R**
5. При неверной клемме — подсказка; при верной — провод фиксируется на винте.
6. Когда подключены все 4 провода — уведомление о завершении уровня.

## Структура

```
public/
  models/thermostat.glb         # 3D-модель термостата (из ТЗ)
  textures/thermostat_img0.png  # diffuse-текстура, извлечённая из .glb (см. ниже)
src/
  main.ts                       # точка входа
  core/
    Game.ts                     # сцена, камера, свет, raycasting, стейт-машина уровня
    layout.ts                   # раскладка клемм и проводов (цвет → клемма)
    Tween.ts                    # мини-движок анимаций без зависимостей
  objects/
    Wall.ts                     # стена (процедурная текстура)
    BackPanel.ts                # задняя панель (процедурная: плита + canvas-фейсплейт + клеммы)
    Thermostat.ts               # загрузка/нормализация .glb, анимация «снятия»
    Wire.ts                     # провод как трубка вдоль кривой Catmull-Rom
  ui/
    Hud.ts                      # баннер, чек-лист проводов, тосты, оверлей завершения
    styles.css
```

## Что сделано «самостоятельно» (по ТЗ — кодом)

В ТЗ был дан только `thermostat.glb`; задняя панель, стена и провода — мои:

- **Задняя панель** построена процедурно: скруглённая плита (`ExtrudeGeometry`) +
  плоский фейсплейт (`PlaneGeometry` с чистыми UV), нарисованный в `<canvas>` —
  две колонки клемм `S S Y Y₂ G C U U` / `A A W₂ E W K R Rc`, золотые винты вплотную
  к буквам, центральный слот для проводов, надпись `U & R WIRES`. Пиксельная раскладка
  канваса считается из тех же 3D-координат, что и точки подключения проводов, поэтому
  винт, подпись и наконечник провода всегда совпадают.
- **Провода** — `TubeGeometry` вдоль кривой Catmull-Rom с медным наконечником;
  подключение анимируется перемещением свободного конца на винт клеммы.
- **Стена** — плоскость с процедурной текстурой краски.

## Примечание по модели

`thermostat.glb` использует устаревшее расширение
`KHR_materials_pbrSpecularGlossiness`, которое современный Three.js не
поддерживает — из-за этого модель грузилась тёмно-серой. Diffuse-текстура
извлечена из `.glb` в `public/textures/thermostat_img0.png` и применяется к
материалу при загрузке (`Thermostat.ts`), что возвращает кремовый цвет и
зелёный дисплей как в референсе.

## Управление

- **ЛКМ по термостату** — снять со стены.
- **ЛКМ по проводу** — выбрать (нужная клемма подсвечивается кольцом).
- **ЛКМ по клемме** — подключить выбранный провод.
- **Drag мышью** — орбита камеры вокруг панели (ограничена).
